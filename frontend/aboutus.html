Team 2 Project Design

Project Overview

Developed by Team 2 as a Computer Engineering design project, this system provides an offline-capable greenware inspection solution designed for future sensor and model integration, enabling early defect detection and improved quality control in local manufacturing.

🎯 Our Mission

To create an affordable, non-destructive inspection system that empowers small-scale pottery manufacturers through accessible and intelligent technology.

🌱 Our Vision

To modernize traditional greenware production by integrating sensor-based analysis, machine learning, and digital twin visualization.

🔍 Key Capabilities

  • Surface and subsurface defect detection
  • Multimodal sensor fusion (thermal, optical, structural)
  • Offline-capable system for local deployment
  • Future-ready for Raspberry Pi and sensor integration
  • Digital twin visualization for real-time monitoring

⚙️ Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python (FastAPI)
  • AI/ML: Deep Learning Models (CNN / RT-DETR ready)
  • Deployment: Raspberry Pi (planned)
